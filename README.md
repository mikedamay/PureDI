usage:
To run tests:
install dotnetcore 2.0
cd PureDITest

then
On Windows
dotnet test

On Linux
dotnet test -c LinuxTest

On Mac
dotnet test -c MacOsTest

all case sensitive

Sample programs:

LoadTest, PureDIDemo
change to directory and do "dotnet run"

PureDIDocumentor
cd PureDIDocumentor
dotnet run

browse to http://localhost:60653/
